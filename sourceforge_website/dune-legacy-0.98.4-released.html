Dune Legacy 0.98.4 Released

August 13, 2024

I am excited to announce the release of Dune Legacy version 0.98.4 for Windows and macOS! This update brings further performance improvements, bug fixes, and enhanced stability.

Key Improvements in 0.98.4

  • Enhanced graphics performance and rendering optimizations
  • Improved AI behavior and decision making
  • Better memory management and reduced memory usage
  • Fixed various crashes and stability issues
  • Updated SDL2 libraries for better compatibility
  • Improved network play stability
  • Enhanced user interface responsiveness
  • Better support for high-resolution displays
  • Various gameplay tweaks and balance improvements
