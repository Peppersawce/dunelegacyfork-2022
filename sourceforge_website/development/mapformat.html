Map Format

Dune II uses INI-Files for describing scenarios. The format is used as a basis for custom maps in Dune Legacy. The following document describes the format used by Dune II and how it is extended. The extensions are not specific to Dune Legacy but shall be implementable by every Dune II clone. If you have any ideas how to extend the format contact us.

  • Dune II Scenario Format Specification - Version 0.4.0 (18.07.2011)
  • 2P - 64x64 - Skeleton Map.ini - A draft for creating a 64x64 map
  • 6P - 128x128 - Skeleton Map.ini - A draft for creating a 128x128 map
