Dune Legacy 0.97.04 Released

April 13, 2025

I am pleased to announce that Dune Legacy version 0.97.04 is now available for Windows! This release brings significant performance optimizations and gameplay improvements.

Key Improvements in 0.97.04

  • Optimized sandworm behavior to use AMBUSH mode for better performance
  • Improved coordinated attack behavior with leader-based targeting
  • Fixed sandworm aggression to prevent excessive hunting across the map
  • Overall improvements in CPU efficiency during large-scale battles
  • Reduced CPU load during AI decision-making
